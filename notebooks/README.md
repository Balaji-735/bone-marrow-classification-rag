# Jupyter Notebooks

This directory contains optional Jupyter notebooks for interactive exploration and development.

## Notebooks

- **01_eda.ipynb**: Exploratory Data Analysis - Dataset exploration, class distribution, sample image visualization
- **02_train_vit.ipynb**: Interactive ViT Training - Train the model with custom parameters and visualize training history
- **03_explainability_demo.ipynb**: Explainability Demo - Generate and visualize Grad-CAM and attention maps

## Usage

To use these notebooks:

1. Install Jupyter: `pip install jupyter`
2. Start Jupyter: `jupyter notebook`
3. Navigate to the notebooks directory and open the desired notebook

Note: These notebooks are optional. All functionality is available through the main.py script and Streamlit dashboard.
